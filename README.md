# Laravel Sanctum SPA
Template dasar untuk membangun projek Laravel Sanctum SPA.

## Fitur
* Laravel 7
* Vue + Vue-router + Vuex
* Autentikasi dengan Sanctum
* Bulma + Buefy + Font Awesome 5 or Material Design Icons [docs Buefy](https://buefy.org/documentation/start)
